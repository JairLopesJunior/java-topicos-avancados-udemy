Optional

- Classe pertencente ao pacote java.util, utilizada como um container de objeto que pode ou não
ter valor nulo;

- Lidar com a exceção NullPointerException;

- Optional não verifica se é null ou não é null, ele verifica se contem ou não um determinado objeto.

- Optional não implementa a interface Serializable, logo o objeto a partir desta classe não pode ser serializado.

- A classe Optional não possui construtores publicos, somente metodos estaticos.
